# Vuex
[Vuex](https://v3.vuex.vuejs.org/zh/)   

## state
- 单一状态
- 响应式
## mutations
- 必须由commit触发
- 必须是同步函数

## getter
- 由state派生出的转态
- store的计算属性

## actions
- 由dispatch触发
- 可以包含任意异步操作
- 不可用变更状态，可以已通过mutation改变
