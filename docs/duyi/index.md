- 全局事件总线$bus
`Vue`强制更改了`VueComponent`的原型对象指向`Object`的原型对象的隐式链，将其改到指向`Vue`的原型对象上。
> VueComponent.prototype.__proto__ === Vue.prototype
> `VueComponent`的原型对象就是`Vue`的原型对象，所以组件实例对象（`vc`）即可访问到 `Vue`原型上的属性、方法。
```js
new Vue({
    el: '#app',
    render: h => h(App),
    beforeCreate(){
        // 安装全局事件总线，$bus就是当前的vm实例,vm上有$on,$emit,$off，其他组件就能访问到了
        Vue.prototype.$bus = this;
    }
});
```
